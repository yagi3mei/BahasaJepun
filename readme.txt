１．開発背景
　SMAP kajangは私が2022年に訪問した学校で、特に2年生で日本語というか、
　勉強が嫌いだが、ITは好きなA君のために作成した。

２．事前作業
　・Quizletで作成した単語カードの学習用であり、単語カードは別にQuizlet上で
　　作成する必要ある。

　・作成した単語カードのエンベットのURLをscript.jsに反映させる必要あある。

　・学習するためにはQuizletへのログインが必要である。


２．気になること
　Quizlet側のアクセス制限が働くことがあり、利用できない時間が発生する。
　　→その場合は時間をおいてからアクセスすること。

　なので、学校で利用する場合は、Wifiのグローバルアドレスを覚えているかもしれない
　ので、一度に多数の生徒がアクセスするとアクセス制限に掛かる可能性があるが、
　現在のところ検証していないので、よくわからない。
　とにかく、先生に送信してやってもらおう！
